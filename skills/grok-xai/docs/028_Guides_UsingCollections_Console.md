# Using Collections in Console

This guide walks you through managing collections using the [xAI Console](https://console.x.ai/) interface.

* * *

## [Creating a new collection](https://docs.x.ai/docs/guides/using-collections/console#creating-a-new-collection)

1.  Navigate to the **Collections** tab in the [xAI Console](https://console.x.ai/).
2.  Ensure you are in the correct team.
3.  Click on "Create new collection".
4.  Optionally enable "generate embeddings on document upload" (recommended).

* * *

## [Viewing and editing collection configuration](https://docs.x.ai/docs/guides/using-collections/console#viewing-and-editing-collection-configuration)

You can view and edit the Collection's configuration by clicking on **Edit Collection**. This allows you to view the current settings and make changes via a modal.

* * *

## [Adding a document to the collection](https://docs.x.ai/docs/guides/using-collections/console#adding-a-document-to-the-collection)

1.  Click on a collection in the table to view its documents.
2.  Click on "Upload document" to add a new file.
3.  Once uploaded, each document is assigned a **File ID**.
4.  View the File ID, Collection ID, and document hash by clicking on the document in the table.

* * *

## [Deleting documents and collections](https://docs.x.ai/docs/guides/using-collections/console#deleting-documents-and-collections)

Use the "more" (three dots) button on the right side of any collection or document row to perform deletion.
